# FlyingHouseVR - PLAY NOW

## Quick Start (30 seconds)

1. Clone repo
2. Open Unity 2022.3
3. Open project folder
4. Open scene: `Assets/Scenes/QuickStart.unity`
5. Press **Play**
6. **You are flying.**

## Controls

### Desktop
- **WASD** - Steer and Throttle
- **Space** - Brake/hover
- **Mouse Right+Drag** - Look around
- **1-4** - Jump to cities (NYC, Paris, Tokyo, SF)
- **F1-F4** - Weather (Clear, Cloudy, Rain, Storm)
- **R** - Open Room Menu (if unlocked)
- **Esc** - Pause

### VR (Quest)
- **Grip** - Grab wheel or lever
- **Trigger** - Interact
- **Menu** - Pause
- **Thumbstick** - Move (if enabled)

## Build for Quest

1. Switch to Android platform
2. Add your Cesium token (Project Settings > Cesium)
3. Build → Deploy to headset

## Troubleshooting

**"Pink terrain"?** → Cesium token not set. Get free token at cesium.com/ion.

**No VR input?** → Check XR Plug-in Management, enable OpenXR + Oculus.

**Stuck?** → Press Escape, click Quit, reload.
